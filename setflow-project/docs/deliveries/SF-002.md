# SF-002 — Configurar arquitetura, DI e navegação

## Estado
DONE

## Contexto
`SF-001` criou um app Android mínimo em Compose. Esta entrega estabelece os containers iniciais para evoluir o app sem conectar UI diretamente a fontes de dados.

## Objetivo
Criar a estrutura base de pacotes, injeção de dependência e navegação tipada com destinos iniciais.

## Pré-condições
- `SF-001` concluída.
- Projeto compila com `assembleDebug`.

## Escopo obrigatório
- Criar `Application` Hilt.
- Configurar Hilt no módulo Android.
- Criar `AppClock` injetável.
- Criar provider de dispatchers.
- Criar estrutura inicial de pacotes `app`, `core`, `feature` e `navigation`.
- Criar navegação base com destinos `Home`, `Program`, `History` e `Settings`.
- Exibir barra inferior fora do treino ativo.
- Usar logo Repcore fornecida como asset inicial.

## Fora do escopo
- Room, DataStore, seed e backup.
- ViewModels de feature.
- Fluxo de treino ativo.
- CRUD de exercício/modelo.
- Navegação profunda com argumentos reais.

## Regras de implementação
- UI não acessa DAO ou fonte de dados.
- Destinos de navegação devem ser tipados.
- Dependências devem ficar fixadas no version catalog.
- Não declarar permissão `INTERNET`.

## Arquivos ou áreas esperadas
- `app/src/main/java/com/douglas/repcore/app/`
- `app/src/main/java/com/douglas/repcore/core/`
- `app/src/main/java/com/douglas/repcore/feature/`
- `gradle/libs.versions.toml`
- `app/src/main/AndroidManifest.xml`

## Critérios de aceite
1. App possui `Application` anotada para Hilt.
2. `MainActivity` é ponto de entrada Compose com Hilt.
3. Rotas base são navegáveis por barra inferior.
4. `AppClock` existe com implementação Android injetável.
5. Dependências novas estão no version catalog.
6. `assembleDebug` passa.

## Testes obrigatórios
- Build debug.
- Unit test task.
- Inspeção do manifest para ausência de permissões remotas.

## Comandos de validação
```bash
./gradlew assembleDebug
./gradlew testDebugUnitTest
```

## Resultado
- Estrutura base criada.
- Hilt, Navigation 3 e coroutines configurados.
- Logo Repcore adicionada ao app.
